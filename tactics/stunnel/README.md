Can [contact them](http://www.stunnel.org/?page=contact) for non-GPL licences.

Supports a set of predefined protocols for which it can setup secure tunnels
using OpenSSL. Supported protocols are:

- cifs
- connect
- imap
- nntp
- pgsql
- pop3
- proxy
- smtp
